# MENSA試験申込 手順書

## LINE通知が来たら

### ▶ 最速の方法（推奨）
デスクトップの **「申込.bat」** をダブルクリック

→ 自動で候補日程とGoogleカレンダーの空き状況が表示される
→ 番号を入力して選択
→ ブラウザが起動してフォームが自動入力される
→ CAPTCHAの数字を手動入力して「送信」ボタンを押す

---

### ▶ コマンドで実行する場合

```
cd "C:\Users\shing\OneDrive\ドキュメント\014_DEV\20260419_MENSAテスト通知"
python register.py
```

特定の試験IDを直接指定する場合（LINE通知のURLから確認）:
```
python register.py <試験ID>
例: python register.py 715
```

## 仕組みの概要

| 役割 | 内容 |
|---|---|
| **GitHub Actions** | 5分おきにMENSAサイトを監視（24時間稼働） |
| **monitor.py** | 関東の新枠・空き枠を検知してLINEに通知 |
| **register.py** | Googleカレンダー照合 + フォーム自動入力 |
| **google_calendar.py** | Googleカレンダー連携ヘルパー |

---

## ファイル構成

```
20260419_MENSAテスト通知/
├── monitor.py          # 監視スクリプト（GitHub Actionsで実行）
├── register.py         # 申込スクリプト（ローカルで実行）
├── google_calendar.py  # カレンダー連携
├── 申込.bat            # ダブルクリックで申込起動
├── config.json         # 個人情報（gitignore済み）
├── credentials.json    # Google API認証情報（gitignore済み）
├── token.json          # Googleトークン（自動生成・gitignore済み）
├── state.json          # 試験枠の監視状態（GitHub Actionsが更新）
└── .github/workflows/
    └── monitor.yml     # GitHub Actionsの設定
```

## 注意事項

- **受験は生涯3回まで**（不合格から次の受験まで1年以上必要）
- 申込締切は試験日の**10日前**
- 申込後は自動返信メールが届くことを確認すること
- `config.json` は個人情報を含むため絶対にGitHubにアップしない
